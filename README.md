# AI-skill

按市场分区存放可打包的 Agent Skills，便于后续扩展：

```
AI-skill/
├── turkey/          # 土耳其 / BIST
├── taiwan/          # 台湾 / TWSE
└── us/              # 美股（预留）
```

每个 skill 仍是独立目录，可单独复制给同事使用。
