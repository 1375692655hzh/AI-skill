# Quickstart

## 开箱即用（3 步）

```bash
pip install -r requirements.txt
$env:MINIMAX_API_KEY = "your-key"
python scripts/generate_info_technical_report.py --config config.json
```

## 推荐执行时间（北京）

```
17:20 首发 → 17:30/40/50, 18:00/10 重试 → 20:00 兜底
```

有 Turkey-investment 缓存时，在 `config.json` 设 `use_project_fetcher: true`。
