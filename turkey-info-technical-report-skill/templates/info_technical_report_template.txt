## 输出格式铁律（违反任何一条即为不合格）

1. 章节标题用【】；技术数据部分**必须使用 Markdown 表格**呈现，这是本报告的核心形式。

3. 禁止 Markdown 加粗/斜体：正文不用 ** 或 __（表格内除外）。
4. 禁止来源署名：正文不得出现任何平台、券商、研究机构、网站、报告名称。
5. 数字精度：点位、涨跌幅、指标值必须与素材一致，不得编造。
6. 篇幅：核心观点与解读 200–400 字；表格部分完整保留素材中的关键数据。
7. 表格规范：使用标准 Markdown 表格（| 列 | 列 |），表头行下方用 |---|---| 分隔。
