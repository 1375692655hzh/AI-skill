# 台股内参 文风与结构规范（已固化 2026-07-14）

本文件为盘前/盘后成稿的**唯一风格基准**，generate.py 的 prompt 依此编写。改风格先改这里，再同步 prompt。
范文样板：`reports/2026-07-14/台股盘后内参_2026-07-14.md`（用户微调定稿版）。

## 通用文风（盘前盘后共用）—— 本质是「做减法」
- 机构投研冷峻客观，事实与判断先行。**不加口语/戏剧化点评**：禁「真金白银在减仓」「独木难支」「性价比偏低」「不是口头看淡而是…」这类评述。
- **删学术黑话与自证话术**：禁「证据链清晰」「结构主导」「当前最高证据等级」「揭示性证据(坐实)」；正文禁「闭环校验通过」「分项和=合计」(校验只在台账)。**事实列够即止，不补「所以证据一致/证据充分」这类自我总结句**(读者自己能看出)。
- **没信息增量的内容直接删**，连「XX持平/无方向性意义/此处从略」这类说明句都不写(写「无意义」本身就是废话)。
- 长句适度拆成短自然段，一句一个意思；判断给到位即可，不铺陈、不堆形容词。
- 少用内部黑话：「侦测双阈值」→「观测点」、「低接」→「低吸」。
- 章节标题不带内部标签(「纯事实/机制层/≤N字」)，也不带括号副标(如「市场表现与传导（隔夜锚）」去掉括号注)。
- **去「制作痕迹」，读者只看到成品**：①不暴露实现细节——ETF代理写「道指/标普500/纳指+X%」不写「ETF代理」、费半写「费半(SOXX)」不写「代理」(来源留台账)；②不写自我指涉/流程说明——禁「见下」「详细拆解见昨日盘后内参」「承接盘后交接/handoff」；③不写已成套话的免责句——如「不构成硬边界/不宜机械解读为必攻涨停」，异常情况一句点明成因即可。
- 禁用词表(lexicon.py)：绞肉机/割韭菜/护盘/杀跌/神救援/腰斩/起飞/庄家/韭菜等。
- 数字只用数据简报，简报无则静默省略或定性，**禁编造、禁占位**。
- 用词：道指(不「道琼」)、影响(不「具体传导」)、关键位(不「关键位网格」)、低吸(不「低接」)、观测点(不「侦测双阈值」)、台股期指(夜盘统一叫法,不「开盘锚/夜盘期指」)。**避免军事化/战斗用词**(「攻击目标」→「定价目标」)。**个股名去 `*` 号**(国巨*→国巨)。删冗余词(「评价有支撑」→「有支撑」)。

## 盘后内参结构（四层金字塔，不带「一二三四」序号）
- 🚩 **核心研判**：2-3 短自然段(方向定性→资金面→关键位与操作姿态)，简洁勿长句堆叠。
- 📊 **大盘**：走势(开高低收/振幅/长下影等)+**均线并入走势段不单列重复**(收盘 vs MA5/10/20、距月线%、空头/多头排列、能否N日收复)+量能+市场宽度+除权息旺季(除息蒸发 vs 含息报酬指数)。段末「📊 **大盘综评**：…」加粗。
- 💰 **资金**：三大法人分项+外资连续天数；期货段标题「**股指期货**」(不写「期货部位(揭示性证据)」)；法人连续买卖超个股三行——🟢外资连买/🔴外资连卖/🔵投信连买(台股红涨绿跌，此处买绿卖红)；土洋对做；汇率。**融资融券完全不提**(静默、不入台账降级)。段末「💰 **资金与筹码综评**：…」加粗。
- 🏭 **行业**：领涨行前🔴、领跌行前🟢；**变动小(约±0.5%内)的板块不单列，挑2-3代表方向、类似板块合并**，不逐一罗列；高低切(未含电子vs未含金融)。段末「🏭 **行业综评**：…」加粗。
- 🎯 **个股异动**：一股一句结论+驱动(法人买卖/营收/除息填息)；法说营收日程简报无则静默不提。段末「🎯 **个股综评**：…」加粗。
- **不含次日展望/盘前部署**(归次日盘前)。
- 🔢 **数字—来源台账**：分层(官方一手/二手同日/派生)、闭环校验只此处列、降级项(不含融资融券)。

## 盘前早评结构（7模块，无板块A/B分隔；标题「# 台股盘前早评（date）」）
**定位=今日作战部署，正餐在「隔夜+夜盘+今日部署」；昨日只做速览、不重复盘后已详述的大盘/筹码复盘。** 范文样板：`reports/2026-07-14/台股盘前早评_2026-07-14.效果版.md`（用户微调定稿）。
- 🚩 **核心结论**(≤180字，最后一句加粗操作姿态)：方向定性→隔夜性质→关键位→操作姿态。
- 📈 **市场表现与传导**(盘前正餐,盘后没有)：隔夜行情(美股四大直接写道指/标普500/纳指涨跌不提代理·费半SOXX点明分化/齐杀·成分股·台系ADR·**台指期夜盘**·油金·台币昨夜锚) → **对台股今日的影响**(分点落到具体个股,不机械1:1映射) → 台积电ADR隐含方向(升水一句点成因,不写硬边界免责套话) → **开盘锚(优先用夜盘)**。长句用句号拆短。
- ⚙️ **关键驱动**(正餐)：其一/其二/其三机制层,每驱动展开一次。
- 🖥️ **交易台观点**(约180字,可拆2段)：隔夜资金本质+对今日开盘拉抬还是抽离+操作含义。**位置在关键驱动之后、昨日速览之前**(隔夜/展望内容连一起,再进昨日+部署)。
- 📊 **昨日速览**(标题就叫「昨日速览」)：压缩,大盘一段(收盘/量能/MA20一句)+筹码一段(法人/连卖天数/期指净空/融资券T-1/涨跌家数);**不写「详见盘后内参」指引句**;日期非紧邻写实际日期。
- 🎯 **盘前策略**(正餐)：内容延续昨日基调但**正文直接写结论(如「策略维持昨日防御风格」),禁「承接T-1/承接handoff/以隔夜数据刷新」流程句**;战术姿态(仓位上限)/分层应对/**关键位**(不叫网格,列上下点位)/回避清单/日程风险(行事历除权息)。
- 🔭 **观测与情景**：观测点(外资卖超阈值/期指净空增减)+两分支情景(多方/空方各附触发确认)。
- 🔢 **数字—来源台账**：分层(含美股ETF代理来源在此标注)+闭环列式+口径差并列;盘前融资券T-1是正常数据不算降级。
